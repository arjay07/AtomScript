Web Version

To get started using the web version, all you need to do is include AtomScript into the head element. The specific tag you include into your head is: <script src="https://rawgit.com/arjay07/AtomScript-Web/master/bin/AtomScript.js"></script>

Then you open a new file and name it main.atom and write your code in there. When you want to test it, you want to add this set of lines inside your head element:
<script>
     AtomScript.src = <This is where you put in the location of your main.atom>;
</script>
For example:
<script>
     AtomScript.src = "scripts/main.atom";
</script>
