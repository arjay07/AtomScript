What is AtomScript?

AtomScript is an interpreted programming language that was created to make programming easier to learn. With simple and efficient syntax, you'll be learning how to create programs and dynamic web pages in no time. The syntax was especially designed for effieciency and the methods of AtomScript were designed to make programming easier to learn than ever.

What are the benefits?

Using AtomScript has many benefits that could contribute to your programming career. This programming language takes something familiar and combines it with something new! AtomScript is generally cross-platform (in development) and it allows developers to use to create websites dynamically. There are so many things AtomScript can do!
